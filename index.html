Pumping Lemma Demonstration Tool

A simulator for disproving regularity.

Formal Definition

For any regular language L, there exists a pumping length p such that any string w ∈ L with |w| ≥ p can be split into w = xyz with:

  • 1. |xy| ≤ p
  • 2. |y| > 0
  • 3. For all i ≥ 0, xyⁱz ∈ L
