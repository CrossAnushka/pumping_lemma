# PUMPING-LEMMA Demonstration Tool for Regular Languages

Designing an educational tool that visually explains the pumping lemma for regular languages. 
The application allows the user to input a language example and a sample string, then the string is divided into three parts (x, y, z). 
The system illustrates how repeating the substring y multiple times affects membership of the string in the language. 
The visualization will help learners understand how the pumping lemma is used to prove that certain languages are not regular.
